如何贡献

感谢您对本网站的关注与改进建议！欢迎通过 GitHub 贡献内容。

1. Fork 仓库

在 wjswjsss.github.io 仓库右上角点击 Fork，创建您自己的副本。

2. 创建新分支

在您的 Fork 中创建一个有描述性的分支，例如 add-contribute-page 或 fix-sidebar-padding。

3. 进行修改

  • 编辑或添加 HTML、CSS 或内容文件。
  • 保持与现有样式和缩进一致。
  • 本地测试：在浏览器中打开 index.html 或 contribute.html。

4. 提交并推送

使用清晰的提交信息提交更改，然后推送到您的分支：

git add .
git commit -m "添加 如何贡献 页面"
git push origin your-branch-name

5. 创建 Pull Request

访问原仓库并点击 New Pull Request，选择您的分支并描述修改内容。

6. 审查并合并

经过审核后，Pull Request 将被合并。感谢您的宝贵贡献！

问题反馈

可使用 GitHub Issues 报告 Bug 或提出功能建议。

其他方式

如果您不了解如何通过 GitHub 贡献，请发送邮件至 wangjiashushenzhen@gmail.com， 附上包含您想添加的内容或想分享的经验的 PDF、MD 或 DOC 文件，我们会帮您发布在页面上。
